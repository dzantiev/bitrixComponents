# Список элементов
Пример использования
```php
$APPLICATION->IncludeComponent('odva:elements', 'temp'//template -- имя шаблона компонента
	,[
	'filter' => [
		'IBLOCK_ID' => 5,//номер инфоблока
		'ACTIVE'    => 'Y',//указавает что доставать из активного инфоблока
		//параметры передаются в GetList филтр
	],
	'sort' =>
	[
		'ID' =>'desc'
		//параметры передаются в GetList сортировку
	],
	"TEMPLATE_PAGN" =>"pagn",//шаблон для пагинатора, если его не указать то массив с пагинатором не выведется
	"PAGE" =>$_GET['PAGE'],//страница которую вы хотите достать из выборки
	"count" => 2 // количество элементов которые хотите достать из инфоблока или будет на странице
]);
```
