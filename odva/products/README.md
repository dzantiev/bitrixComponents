#Компонент продукты(список продуктов)

- Пример подключения
```php
	$productsResult = $APPLICATION->IncludeComponent(
		"odva:products",
		"catalog",
		[
			'filter' =>
			[
				//все параметры попадают в параметр filter метода getlist
				'IBLOCK_ID'    => 2//id каталога
			],
			'sort'   =>
			[
				//все параметры попадают в параметр order метода getlist
				'ID'=>'desc'// сортровка результа
			],
			'count'  => 10, //кодичество доставаемых елементов
			'page'   => 1 //страница которую выдостаете(этот параметр важен при реализаций пагинации для компонента)
		]
	);
	//либо(акутуально для актуального шаблона)
	$APPLICATION->IncludeComponent(
		'odva:products',
		'saleleader',//шаблон подключаемого компонента
		[
			'filter' => [
				//все параметры попадают в параметр filter метода getlist
				'IBLOCK_ID'  => 2,//id каталога
				'ACTIVE'     => 'Y',//доставать только активные товары
				'PROPERTY_SALELEADER_VALUE' => 'Y',//кастомное свойство по которому хотим фильтровать, в данном случае свойство SALELEADER
			],
			'count' => 10,//количество
			'heading' => 'Популярные товары'//заголовок (доп свойство не влияющее на работу компонента)
		]
	);
```
- saleleader - акутуальный шаблон выводящий список товаров
