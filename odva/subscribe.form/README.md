# Компонент форма для подписки на рассылки

- пример подключения
```php
   $APPLICATION->IncludeComponent(
			'odva:subscribe.form',
			'',// имя шаблона
			[
				"format" => "html", //тип подписки "html" или "text", должно совпадать с типом рассылки
				"rub_id" => 45 //id рассылки
			]
		);
```
